# Front-end (Next.js)


Form Validation with React Hook Form & Yup (Next.js)


## Getting Started

Clone the project

```bash
  git clone https://github.com/hattaalfaritzy/form-validation
```

Go to the project directory

```bash
  cd assignment-login
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
